### 프로젝트 소개

**폼나는싸패**는 간단한 설문부터 프로젝트 인원 모집, 스터디 모집 등 다양한 설문을 진행할 수 있도록 설문지 제작 및 응답 서비스를 제공하는 프로젝트입니다.
